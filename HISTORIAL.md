# Historial del proyecto

Este archivo documenta la evolucion real de la Calculadora de Muestreo. Sirve como respaldo narrativo del trabajo realizado, incluyendo periodos donde el desarrollo se hizo de forma local u offline y no quedo registrado diariamente en Git.

## Evidencia disponible en Git

El repositorio conserva actividad real en las siguientes fechas:

| Fecha | Evidencia | Descripcion |
| --- | --- | --- |
| 2025-10-14 | Commit inicial publicado en GitHub | Primeros archivos del proyecto cargados al repositorio. |
| 2025-10-21 | Ajustes y reemplazo de archivos | Limpieza de archivos iniciales y nuevas subidas del prototipo. |
| 2025-10-23 | Nueva iteracion del proyecto | Actualizacion de archivos del prototipo web. |
| 2025-10-25 | Version mejorada publicada | Reemplazo de archivos y consolidacion de la version previa. |
| 2026-06-17 | Reconstruccion en Angular | Reimplementacion como aplicacion Angular con arquitectura por componentes. |
| 2026-06-17 | Documentacion y publicacion | Mejora del README y build preparado para GitHub Pages. |

## Linea de tiempo funcional

### Octubre de 2025: prototipo inicial

- Construccion de una primera version web orientada al calculo de muestras.
- Trabajo con formulas estadisticas para poblaciones finitas e infinitas.
- Ajustes iterativos de archivos HTML, CSS y JavaScript.
- Publicacion progresiva de archivos en GitHub durante octubre de 2025.

### Periodo de trabajo local/offline

- El proyecto fue trabajado y revisado localmente sin commits publicos diarios.
- Este periodo corresponde a practica, ajustes de interfaz y validacion de formulas antes de la reconstruccion moderna.
- Al no existir commits diarios en Git para ese periodo, se documenta aqui como historial de trabajo local reportado por el autor.

### Junio de 2026: version Angular

- Reconstruccion completa del proyecto como aplicacion Angular.
- Separacion de responsabilidades en componentes, modelos y servicios.
- Implementacion de calculos para:
  - tamano de muestra;
  - muestreo estratificado;
  - muestreo por conglomerados;
  - muestreo sistematico;
  - historial local de calculos.
- Preparacion del build para publicacion en GitHub Pages.

## Nota de transparencia

Este historial no modifica fechas de commits ni simula actividad en GitHub. Su objetivo es explicar con claridad el desarrollo real del proyecto y diferenciar entre trabajo registrado en Git y trabajo realizado localmente.
